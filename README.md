## Aperçu local

Ouvre simplement `index.html` dans ton navigateur.

## Déploiement GitHub Pages

1. Va dans **Settings** du repo GitHub.
2. Ouvre la section **Pages**.
3. Dans **Source**, choisis **Deploy from a branch**.
4. Sélectionne la branche **main** et le dossier **/ (root)**.
5. Enregistre.

Ton site sera ensuite disponible sur :

`https://underscore007.github.io/portfolio-cv/`
